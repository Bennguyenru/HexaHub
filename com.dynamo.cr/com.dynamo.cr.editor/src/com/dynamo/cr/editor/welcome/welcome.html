Welcome to Defold

First Step

The fastest way to get started is to run the introductory Side scroller tutorial.

If you want to start by finding your way around on your own, follow these steps:

  1. Create a project in the Dashboard.
  2. Open the project (Ctrl+O or Cmd + O).
  3. Open your files either from the Project Explorer to the left or by browsing using Ctrl+Shift+R or Cmd+Shift+R.

When you edit and save a file, it will appear in the Changed Files-view to the lower left. You can revert the changes using the context menu in that view. The contents of the file are often displayed in the outline to the upper right. The properties of whatever is selected is displayed to the lower right.
